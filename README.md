## Privacy Policy

**Last updated: April 19, 2026**

**Sudoku does not collect, store, transmit, or share any personal data.**

The app functions entirely on-device. No analytics, tracking, advertising, or third-party SDKs are used.

### What is stored

Game preferences (difficulty, sound, strict mode) are stored **locally on your device** using iOS standard storage (UserDefaults) and are never sent anywhere.

### What is not collected

| Data | Collected |
|------|-----------|
| Personal information | ✗ |
| Location | ✗ |
| Usage analytics | ✗ |
| Crash reports | ✗ |
| Advertising identifiers | ✗ |
| Any network requests | ✗ |
